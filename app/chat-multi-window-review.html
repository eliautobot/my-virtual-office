Multi-Chat Window Review

This page is a lightweight reviewer surface for the shared chat UI. Open it with ?chatView=all&chatAgents=auto to force all four windows open and assign different agents to each one.

  • Primary + 3 secondary windows are rendered by the same chat.js.
  • Each window owns its own selected agent/session state.
  • Selections persist per window after explicit changes.
  • Message/history routing stays scoped by window session/run ownership.
