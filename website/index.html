FAQ

What do I need to run this?

Docker and an OpenClaw installation. That's it. Works on Linux, Mac, and Windows (WSL).

Is this a SaaS?

No. It's self-hosted software you run on your own machine. Your data never leaves your server. No accounts, no telemetry.

How do updates work?

docker pull and restart. Your config and customizations are stored in a mounted volume — updates don't touch them.

Can I try before buying?

Yes. The free demo includes 3 agents, chat, presence, the editor, and the model manager. No key required.

What's the difference between Early Bird and Full?

Nothing — same product, same features, same updates. Early Bird is just a limited-time launch discount. Get it while it lasts.

What is OpenClaw?

OpenClaw is an open-source AI agent framework. Virtual Office is a visual layer that runs on top of it. Learn more →
